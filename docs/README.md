## ❓ 常见问题

### 安装和配置

**Q: 如何安装 Cleanup CLI？**

A: 参考 [QUICKSTART.md](../QUICKSTART.md) 的安装部分。

**Q: 如何配置 AI 服务？**

A: 首次运行会启动设置向导，或手动编辑 `~/.cleanuprc.yaml`。

**Q: 支持哪些 AI 服务？**

A: 支持 Ollama（本地）和 OpenAI（云端）。

### 使用问题

**Q: 如何撤销操作？**

A: 使用 `cleanup undo` 命令撤销最后一次操作。

**Q: 如何排除特定文件？**

A: 使用 `--exclude-ext`、`--exclude-pattern` 或 `--exclude-dir` 参数。

**Q: 如何查看操作历史？**

A: 使用 `cleanup history` 命令。

### 开发问题

**Q: 如何运行测试？**

A: 使用 `go test ./...` 运行所有测试。

**Q: 如何添加新的规则类型？**

A: 参考 [internal/README.md](../internal/README.md) 的扩展指南。

**Q: 如何贡献代码？**

A: 提交 Pull Request 到 GitHub 仓库。
